          Concrete QUIC and HTTP/3 event definitions for qlog
            draft-marx-quic-logging-event-definitions-latest

Abstract

   This document describes concrete qlog event definitions and their
   metadata for QUIC and HTTP/3-related events.  These events SHOULD be
   embedded in the higher level schema defined in draft-marx-quic-
   logging-main-schema.
